Software Engineer (Machine Learning)

As a software engineer you will work on our machine learning platform and implement state-of-the-art algorithms for large-scale machine learning using parallel and distributed computing.

We are looking for talented engineers with an in-depth knowledge of algorithms and data structures as well as parallel and distributed computing.

Skills

  • Highly motivated and passionate about ML and soft ware engineering
  • Experience developing efficient and scalable ML algorithms
  • Experience developing large, scalable systems
  • Solid programming skills in Python, C/C++, or Java (Scala, Clojure)
  • Solid grasp of algorithm design and complexity fundamentals
  • In-depth knowledge and hands on implementation experience with at least one non-trivial ML algorithm

Bonus

  • Experience developing ML algorithms for Deep Learning, Recommender Systems, Natural Language Processing, or Online Learning
  • Experience developing software within a distributed computation framework (e.g.  MPI, MapReduce, Spark, or GraphLab)
